# OS Assignment - Group 8
Dice rolling game

** This code is using Ubuntu terminal to run (WSL Windows Subsystem for Linux - can be downloaded through Microsoft store)
** Make sure to cd the file in order to run

- To run the server, type this in the terminal
1. make clean
2. make
3. ./server

- To run client(player), type this on a seperate terminal (3 players)
1. ./client (username)
